== 语法

=== ここ/そこ/あそこ は 名です。

指示场所时，用ここ/そこ/あそこ所表示的位置关系与これ/それ/あれ相同。

例句：

[literal]
----
▲ ここ    は  デパートです。（这里是百货商店。）
▲ そこ    は  図書館です。（那里是图书馆。）
▲ あそこ  は  入り口です。（那儿是入口。）
▲ あそこ  は  コンビニです。（那里是便利店（远距离）。）
----

=== 名 は 名（场所）です。

[literal]
----
▲ 食堂    は  デパート 7階です。（食堂在百货商店的7层。）
▲ トイレ   は  ここ です。（厕所在这儿。）
▲ 小野さん は  事務所 です。（小野在事务所。）
▲ お手洗い は  デパート2階です。（洗手间在超市的2楼。）
----

=== 名 は どこですか

* トイレはどこですか。あちらです。

+
こちら/そちら/あちら 表示场所，人。是比较有礼貌的说法。 表示人的说法，按礼貌从小到大排序是 この人、この方、こちら、こちらの方
+

* あなたのかばんはどこですか。わたしのかばんはここです。

* すみません、ABC大学はどこですか。

=== 名も 名です

助词“も”基本相当于汉语的“也”。

* ここはJC企画のビルです。
* 李さんも中国人です。（小李也是中国人。）
* あなたも中国人ですか。（你也是中国人吗？）
* あなたも北京大学の学生ですか。（你也是北京大学的学生吗？）

=== 名 は 名 ですか、名ですか

答案有多种可能，而询问其中的一种时，可以重复使用谓语“~ですか”。

* かぶん売り場は一階ですか、二階ですか、三階ですか、四階ですか。
* 今日は水曜日ですか、木曜日ですか。
* 林さんは韓国人ですか、日本人ですか、中国人ですか。
* 中村さんは医者ですか、教師ですか、社長ですか。

=== 名はいくらですか

* これはいくらですか。
* その服はいくらですか。
* このセーターはいくらですか。

== 表达和词语讲解

. 1階
+
1階不读“いちかい”，而是读“いっかい”。以后的课会讲量词的变化。
+
. 省略
+
これはいくらですか、それは？あれは？
+
. 礼貌用语
+
こちら、そちら、あちら、どちら比ここ、そこ、あそこ、どこ更礼貌。有个常用句子，どちらさまですか，表示请问您是哪一位。
+
. 缩略词
+
パソコン、デジカメ、コンビニ
+
. あのう
+
类似英语的“excuse me”。
+
. ~

+
“か”升调表示疑问，“か”降调表示“原来如此”。类似的（升/降调）还有そうですか。
+


== 课文对话

[literal]
----
A: トイレはどこですか。
B: あちらです。
----

[literal]
----
A: ここは  郵便局ですか、銀行ですか。
B: 銀行です。
----

[literal]
----
A: これはいくらですか。
B: それは 5800円です。
A: あれは？
B: あれも 5800円です。
----
