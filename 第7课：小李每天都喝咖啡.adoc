== 语法

=== 名词を动词

动作的对象用助词を表示，这里读作お。

[literal]
----
* 李さんは　毎日　コーヒー　　を　飲みます。
* わたしは　毎日　ジョギング　を　します。
* わたしは　　　　新聞　　　　を　読みません。
* 李さんは　毎朝　何　　　　　を　食べますか。
----

=== 名词【场所】で动词

在某地做某事。存在场所和动作进行的场所在汉语中都用在，日语中存在场所是に、动作进行的场所用で。

[literal]
----
* 李さんは　図書館で　　　　　　勉強します。
* わたしは　コンビニで　お弁当　買います。
* 　　今日　どこで　新聞を　　　読みましたか。
* 昨日、私は王さんと　図書館で　本を　読みました。
----

=== 名词か名词 AかB

对若干名词进行选择是，名词和名词之间加“か”，相当于中文重的“或者”。

[literal]
----
* わたしは　毎朝　パンか　お粥を　食べます。　
*　　休みは　何曜日ですか。
 　　休みは　月曜日か　火曜日です。
* 今晩、私は　アニメか　ドラマを 見ます。
----

=== 名词を　ください

常用于花钱购物，不花钱的索取。 すみません、三番お　ください。

[literal]
----
*　コーラと　ケーキを　ください。
*　　　　　　申込書を　ください。
*　　　　　　この本を　ください。
*　　　チーズケーキを　ください。
----

=== “何”的发音

* 何　なに　＋　と／が／を／で。

** 何と　何を　買いますか。
** あそこに　何が　ありますか。
** 何を　食べますか。
** 何で　会社ヘ　行きますか。

* 何　なん　＋　で（不建议）/の/（时刻/星期）。

** 何で　会社ヘ　行きますか。
** それは　何の　本ですか。
** 李さんは　何時に　きますか。
